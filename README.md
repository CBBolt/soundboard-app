# Soundboard App

This is a simple soundboard app that is built with React, TypeScript, and Electron. It is meant to go with VoiceMeeter and VB Cable to allow for playing the sounds along with Mic audio in games and such.

The actual EXE to use is under the Releases section in the GitHub Repo.

Install instructions and such are included in the app itself to help with onboarding and provide some interactive instructions to help get the user started.
